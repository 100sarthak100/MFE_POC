# MicroFrontend POC

## Run Locally
- Clone the repo.
```
cd app1
npm install
npm start

cd app2
npm install
npm start

cd app3
npm install
npm start

cd app4
npm install
npm start

cd app5
npm install
npm start

cd navbar
npm install
npm start

cd root-html-file
npm install
npm start
```

- Root container application will start at -> http://localhost:4200/Nav1/app1

- Make sure to run the spring boot backend before that -> https://github.com/100sarthak100/SpringBoot_Microservices
